# Password Strength Lab

**Password Strength Lab** is a modern desktop application built with Python and CustomTkinter that analyzes password security in real time. It evaluates password strength using industry-standard complexity checks, visualizes security with a dynamic strength meter, generates smart improvement suggestions, and securely hashes passwords using SHA-256 â€” all inside a sleek, AI-styled purple interface.

---

##  Features

-  Real-time password strength analysis  
-  Dynamic, color-coded strength meter  
-  Intelligent password improvement suggestions  
-  Secure SHA-256 password hashing  
-  Optional secure local password hash storage  
-  Modern dark UI   

---

##  Tech Stack

- **Python**
- **CustomTkinter**
- **Regex**
- **Hashlib**
- **Tkinter**
